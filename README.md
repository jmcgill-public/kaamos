# Kaamos

**First rule of Kaamos:** Do not explain Kaamos.

**Second rule of Kaamos:** Do NOT explain Kaamos.

---

This is a novel, a game, a music project, and a cosmological framework operating in the same semantic space.

The game teaches you the language without announcing that it's doing so. You complete sequences. Glyphs accumulate. You learn to read the world before you read about it.

The novel witnesses rather than judges. Every character knows something incompletely. Every action is right and wrong simultaneously. You hold both.

The music is in F# Phrygian. It has a death whistle in it. The flute arrives after.

Enter at [project documentation](CLAUDE_kaamos.md) if you need context. But the point is not to need it.
